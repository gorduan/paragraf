## Interaction Contracts

### Recommend Button (D-01, D-02, D-03, D-04)

| State | Behavior |
|-------|----------|
| Default | Button "Aehnliche finden" with Sparkles icon in ResultCard action bar |
| Loading | Button disabled, inline spinner replaces icon |
| Expanded | RecommendSection slides open below ResultCard with 3 Mini-Cards |
| Mini-Card collapsed | Single line: "SS 1 SGB IX" + score badge, clickable to expand |
| Mini-Card expanded | Full text visible below header line |
| "Mehr laden" in section | Button below 3 Mini-Cards loads remaining (up to 10 total) |
| Mini-Card actions | Each Mini-Card has: expand/collapse, compare button, bookmark button |
| Collapse | Clicking "Aehnliche finden" again collapses the section |
| Error | Inline error text "Empfehlungen konnten nicht geladen werden" below button |

Animation: RecommendSection expands with max-height transition, 200ms ease-out. Respects prefers-reduced-motion.

### View Toggle (D-05, D-06, D-07)

| State | Behavior |
|-------|----------|
| Default | Persisted from localStorage; falls back to "Liste" |
| Liste active | Flat list of ResultCards (current behavior) |
| Gruppiert active | Accordion with gesetz groups; first group open, rest closed |
| Accordion header | Gesetzname (subheading) + Badge with hit count, click toggles open/close |
| Accordion content | List of ResultCards within that group |
| Toggle position | Right side of results header bar, next to "X Ergebnisse" count |
| Persistence | View mode saved to localStorage key "paragraf_view_mode" |

### Filter Panel (D-08, D-09, D-10, D-11)

| State | Behavior |
|-------|----------|
| Collapsed | "Erweiterte Filter" button with SlidersHorizontal icon below SearchBar |
| Expanded | Panel slides open with filter fields |
| Abschnitt filter | Select dropdown populated from available abschnitt values in results metadata |
| Chunk-Typ filter | Two radio options: "Paragraph" and "Absatz" (maps to chunk_typ API param) |
| Absatz-Range filter | Two number inputs "Von" and "Bis" for absatz range |
| Apply | "Filter anwenden" primary button at bottom of panel; triggers new search with filters |
| Active filters | Chips appear between filter panel and results, each with X to remove |
| "Alle zuruecksetzen" | Text link in destructive color, clears all filters and re-searches |
| Empty filters | No chips shown, no "zuruecksetzen" link |

### Search Mode Toggle (D-14, D-15)

| State | Behavior |
|-------|----------|
| Position | Directly below the SearchBar input row |
| Segments | 3 segments: "Semantisch" (default), "Volltext", "Hybrid" |
| Active segment | primary-500 background, white text |
| Inactive segments | neutral-100 background (dark: neutral-700), neutral-600 text |
| Interaction | Click selects mode; next search uses that mode via search_type param |
| Mapping | Semantisch -> "semantic", Volltext -> "fulltext", Hybrid -> "hybrid_fulltext" |

### Pagination (D-12, D-13)

| State | Behavior |
|-------|----------|
| Initial | No button shown until first search returns results |
| Has more | "Mehr laden" button centered below results; "10 von ~42 Ergebnissen" text above it |
| Loading more | Button shows inline spinner, text changes to "Lade weitere..." |
| No more | Button disappears; text shows "Alle X Ergebnisse angezeigt" |
| Mechanism | Appends new results to existing array; uses next_cursor from API response |

### Compare Wiring (D-16)

| State | Behavior |
|-------|----------|
| No selection | Compare button on each ResultCard shows "Vergleichen" with GitCompare icon |
| Selected | Compare button toggles to "Ausgewaehlt" state with check icon, primary-500 color |
| Counter | Floating indicator or badge in page header: "2 zum Vergleich" with link to ComparePage |
| State management | CompareContext in App.tsx (pattern matches BookmarkContext) |
| Max selection | No hard limit; ComparePage handles display |
| Persist | Not persisted in localStorage (compare is session-scoped) |

## Accessibility Contract

| Element | ARIA / Keyboard |
|---------|-----------------|
| RecommendSection | `aria-expanded` on trigger button; section has `role="region"` with `aria-label="Aehnliche Paragraphen zu {ref}"` |
| View Toggle | `role="radiogroup"` with `role="radio"` + `aria-checked` on each segment |
| Grouped Accordion | Each header is `<button>` with `aria-expanded`; content panel has `role="region"` |
| Filter Panel | `aria-expanded` on toggle button; panel has `aria-label="Erweiterte Filter"` |
| Filter Chips | Each chip is `<button>` with `aria-label="Filter entfernen: {filter}"` |
| Search Mode Toggle | `role="radiogroup"` with `role="radio"` + `aria-checked` on each segment |
| Load More | `aria-live="polite"` on result count text; button has descriptive label |
| Compare counter | `aria-live="polite"` to announce selection changes |
| Mini-Cards | Same `<article>` pattern as ResultCard with `aria-label` |

|------------|----------|
