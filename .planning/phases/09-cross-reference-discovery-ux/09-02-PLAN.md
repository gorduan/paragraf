    ```typescript
    import { forceSimulation, forceLink, forceManyBody, forceCenter, forceCollide } from "d3-force";
    ```

    Functions:
    - `nodeRadius(node: GraphNode): number` — returns 24 + Math.min(node.refCount, 24) for law nodes (range 24-48px diameter), 16 + Math.min(node.refCount, 16) for paragraph nodes (range 16-32px)
    - `createSimulation(nodes: GraphNode[], links: GraphLink[], width: number, height: number): Simulation<GraphNode, GraphLink>` — uses forceLink with id accessor and distance 80, forceManyBody strength -200, forceCenter, forceCollide with radius nodeRadius+5, alphaDecay 0.02 for faster cooling
    - `findNodeAtPoint(x: number, y: number, nodes: GraphNode[], transform: Transform): GraphNode | null` — iterates nodes, converts canvas coords with transform, checks distance <= nodeRadius. Returns first match or null.
    - `drawFrame(ctx: CanvasRenderingContext2D, nodes: GraphNode[], links: GraphLink[], transform: Transform, selectedNodeId: string | null, hoveredNodeId: string | null, isDark: boolean): void` — clears canvas, applies transform, draws dot grid background (20px intervals, neutral-200/neutral-700), draws edges as arrows with EDGE_COLORS[isDark ? "dark" : "light"][link.kontext], draws arrowheads (8px), draws nodes as circles with fill/stroke per UI-SPEC, draws node labels (12px system font, centered).
    - `drawArrow(ctx, fromX, fromY, toX, toY, targetRadius, color, width)` — line + arrowhead as per Research code example
    - `buildLawLevelGraph(lawRefs: Map<string, { outgoing: {gesetz: string, kontext: string | null}[] }>): { nodes: GraphNode[], links: GraphLink[] }` — aggregates per-paragraph references to law-level. Creates one node per law, one link per unique (sourceLaw, targetLaw, kontext) triple. Sets refCount to total reference count for each law node.

    **GraphCanvas.tsx** (per D-07, D-08):

    Props:
    ```typescript
    interface GraphCanvasProps {
      nodes: GraphNode[];
      links: GraphLink[];
      selectedNodeId: string | null;
    ```

    Implementation:
    - `canvasRef = useRef<HTMLCanvasElement>(null)`
    - `simRef = useRef<Simulation>()` — stores d3-force simulation
    - `transformRef = useRef<Transform>({ x: 0, y: 0, k: 1 })` — pan/zoom state
    - `rafRef = useRef<number>()` — requestAnimationFrame handle
    - `dragRef = useRef<{ node: GraphNode; offsetX: number; offsetY: number } | null>(null)` — drag state

    useEffect on [nodes, links]: create simulation via `createSimulation()`, listen to `sim.on("tick")` and call `drawFrame()` in rAF loop. On `sim.on("end")`, cancel rAF. Return cleanup: `sim.stop()`, `cancelAnimationFrame`.

    Canvas DPI scaling (Pitfall 1): on mount and resize, set `canvas.width = rect.width * dpr`, `canvas.height = rect.height * dpr`, CSS dimensions to logical size, `ctx.scale(dpr, dpr)`.

    Mouse events on canvas:
    - `onMouseDown`: check `findNodeAtPoint()`. If node found, start drag (fix node position `fx`, `fy`). If no node, start pan.
    - `onMouseMove`: if dragging node, update `fx`/`fy`, reheat simulation. If panning, update `transformRef`. Call `findNodeAtPoint` for hover detection, call `onNodeHover`.
    - `onMouseUp`: if was dragging, release `fx`/`fy`. If was pan, stop.
    - `onWheel`: adjust `transformRef.k` (clamp 0.3 to 3.0), adjust x/y to zoom toward mouse position. Show zoom level in bottom-right corner.
    - `onClick`: if no drag occurred (mouse didn't move), call `findNodeAtPoint()` then `onNodeClick`.

    Canvas: `role="img"`, `aria-label="Interaktiver Zitationsgraph mit {nodes.length} Knoten"`, `tabindex={0}`.
