## Visual Hierarchy

**GraphPage focal point:** The force-directed graph canvas is the primary visual anchor. At lg breakpoint (>= 1024px) the canvas takes approximately 70% of viewport width (flex-1 minus the 320px side panel when open, full width when panel is closed). All other GraphPage elements (level toggle, legend, zoom controls) are lightweight overlays on or adjacent to the canvas. The side panel is secondary — it appears on demand and never competes with the canvas for visual dominance.

**SearchPage (discovery mode) focal point:** The result list remains the primary visual element. The DiscoveryExampleBar is a compact supporting element (single row height when chips fit, max two rows) positioned between the mode toggle and results. It does not push results below the fold at any breakpoint.

## Interaction Contracts

### Citation Links (D-01, D-02, D-03, D-04, D-05)

| State | Behavior |
|-------|----------|
| Default | Citation text has primary-50 background highlight (dark: primary-900/30), ExternalLink icon (14px) inline after text, context label as Badge variant="default" (12px) showing "i.V.m." / "gemaess" / etc. |
| Hover | Background deepens to primary-100 (dark: primary-800/40), cursor pointer, Radix Tooltip opens after 300ms delay |
| Tooltip content | Truncated paragraph excerpt (max 150 chars) + "Mehr lesen" link. If citation is in LAW_REGISTRY: shows "SS X Abs. Y GESETZ" heading + excerpt. |
| Click (verified) | Navigate to LookupPage with gesetz + paragraph pre-filled. Equivalent to existing lookup flow. |
| Click (unverified) | Show inline warning text "Dieses Gesetz ist nicht indexiert" in warning-500 color below the citation. No navigation. |
| Nested (D-05) | Citations within tooltip excerpts are also CitationLink components. Clicking them navigates (closes current tooltip). |
| Keyboard | Tab-focusable. Enter/Space triggers click. Escape closes tooltip. |

Icon choice: `ExternalLink` from lucide-react at 14px, positioned 2px right of citation text, colored primary-400 (dark: primary-300). Opacity 0.7 default, 1.0 on hover.

### Citation Graph (D-06, D-07, D-08, D-09, D-10)

| State | Behavior |
|-------|----------|
| Initial load | GraphPage loads with law-level view. Each node = one law (e.g., "SGB IX"). Nodes sized by total reference count. |
| Law-level nodes | Circle with 2-letter abbreviation inside. Size: 24-48px diameter based on reference count. Fill: primary-100 (dark: primary-800). Stroke: primary-400. |
| Paragraph-level nodes | Smaller circles (16-32px). Label: "SS X". Fill: neutral-100 (dark: neutral-700). Stroke: neutral-400. |
| Level toggle | Button group "Gesetze" / "Paragraphen" top-left of graph area. "Gesetze" is default. Clicking a law node in law-level also drills into paragraph-level for that law. |
| Drill-down | Click law node -> graph transitions to show only paragraphs of that law + their cross-law references. Back button to return to law-level. |
| Edges | Directed arrows (arrowhead at target). Color by context type (see Graph Edge Colors table). Width: 1.5px default, 2.5px on hover. |
| Pan | Mouse drag on empty canvas area pans the viewport. |
| Zoom | Mouse wheel or pinch gesture zooms. Min 0.3x, max 3x. Zoom level shown in bottom-right corner. |
| Node drag | Mouse drag on a node repositions it. Other nodes adjust via force simulation. |
| Node hover | Node stroke widens to 3px, primary-500 color. Connected edges highlight. Tooltip shows node name + reference counts. |
| Node click | Side panel opens on right (320px width). Graph area shrinks to accommodate. Panel shows: heading, paragraph text excerpt, outgoing count, incoming count, "Im Detail anzeigen" link to LookupPage. |
| Side panel close | X button or click empty graph area. Graph area expands back. |
| Legend | Fixed overlay bottom-left of graph. Shows colored line + label for each context type. Collapsible to icon-only. |
| Empty state | "Keine Querverweise vorhanden. Indexieren Sie zuerst Gesetze mit Querverweisen." |
| Loading | Centered spinner with "Lade Zitationsnetzwerk..." |
| Performance | Progressive loading: show first 50 nodes, "Weitere laden" button for more. Max 200 nodes visible at once. |

Canvas rendering specifics:
- Anti-aliasing: enabled (default canvas behavior)
- Node text: 12px system font, centered, color neutral-700 (dark: neutral-200)
- Background: neutral-50 (dark: neutral-900) with subtle dot grid pattern at 20px intervals
- Arrow heads: 8px length, filled with edge color

### Discovery Search UI (D-11, D-12, D-13, D-14)

| State | Behavior |
|-------|----------|
| Mode activation | User clicks "Entdecken" in SearchModeToggle (4th segment). Checkboxes appear on all ResultCards. |
| Checkbox placement | Left side of ResultCard, before the paragraph reference. Two buttons: green "+" (ThumbsUp icon, 20px) and red "-" (ThumbsDown icon, 20px). |
| No selection | Both +/- buttons are ghost variant (neutral-400 icon). |
| Positive selected | "+" button fills success-500, "-" button remains ghost. |
| Negative selected | "-" button fills error-500, "+" button remains ghost. |
| Toggle | Clicking already-selected button deselects it. Clicking the opposite button switches selection. |
| Example bar | DiscoveryExampleBar appears between SearchModeToggle and results. Shows: green chips for positives ("+ SS 5 SGB IX"), red chips for negatives ("- SS 12 BGB"), each removable with X. "Entdecken" primary button at end. |
| Minimum examples | "Entdecken" button disabled until at least 1 positive example selected. Tooltip: "Waehlen Sie mindestens ein positives Beispiel (+)." |
| Execute discovery | Click "Entdecken" -> calls api.discover() with selected point IDs -> results replace current results below the example bar. |
| Results display | Discovery results shown as standard ResultCards (same as search results). Each can also have +/- buttons for iterative refinement. |
| Reset | "Zuruecksetzen" text link (destructive color) in example bar. On click: immediately clears all selections and restores previous search results, then shows UndoSnackbar at bottom-center with "Beispiele entfernt" message and "Rueckgaengig" action button. Snackbar auto-dismisses after 5 seconds. Clicking "Rueckgaengig" restores all selections and re-applies discovery results. If the snackbar dismisses without undo, the reset is permanent. |
| Mode switch away | Switching from "Entdecken" to another search mode clears discovery selections and restores normal search flow. |
| Max examples | 5 positive + 5 negative (matches API limit). Additional selections disabled with tooltip "Maximal 5 Beispiele pro Kategorie." |

### UndoSnackbar (for discovery reset)

| State | Behavior |
|-------|----------|
| Visible | Fixed position bottom-center, 16px from viewport bottom. Neutral-800 bg (dark: neutral-100), white text (dark: neutral-900). Contains message text + "Rueckgaengig" text button in primary-300 (dark: primary-600). Auto-progress bar shrinks over 5 seconds. |
| Dismiss | Auto-dismisses after 5 seconds, or on explicit close (X icon, 16px). Slide-down exit animation (150ms). |
| Undo clicked | Restores previous discovery state (selections + results). Snackbar closes immediately. |
| Stacking | Only one snackbar at a time. New reset while snackbar visible: dismiss old snackbar (commit that reset), show new snackbar for new reset. |

### Graph Entry from ResultCard (D-16)

| State | Behavior |
|-------|----------|
| Button | "Zitationen" with Network icon (20px) in ResultCard action bar, ghost variant. Visible on all ResultCards that have references_out in metadata. |
| Click | Navigate to GraphPage with query param `?gesetz={gesetz}&paragraph={paragraph}`. Graph centers on that node. |
| No references | Button hidden if paragraph has no outgoing or incoming references. |

### Sidebar Entry (D-15)

| State | Behavior |
|-------|----------|
| Position | After existing nav entries, before Settings. |
| Icon | Network from lucide-react (20px). |
| Label | "Zitationsgraph" |
| Active state | Same indigo accent pattern as other sidebar entries. |
| Keyboard shortcut | Follows existing Ctrl+N pattern (next available number). |

## Accessibility Contract

| Element | ARIA / Keyboard |
|---------|-----------------|
| CitationLink | `role="link"`, `tabindex="0"`, `aria-label="Verweis auf SS {paragraph} {gesetz}"`. Enter/Space triggers navigation. |
| CitationTooltip | Radix Tooltip with `aria-describedby` pointing to tooltip content. Escape closes. |
| Graph Canvas | `role="img"` with `aria-label="Interaktiver Zitationsgraph mit {count} Knoten"`. Focusable with tabindex="0". |
| Graph node focus | Tab cycles through nodes. Enter opens side panel. Arrow keys move between connected nodes. |
| Graph side panel | `role="complementary"`, `aria-label="Details zu {node label}"`. Close button has `aria-label="Seitenpanel schliessen"`. |
| Graph level toggle | `role="radiogroup"` with `role="radio"` + `aria-checked` on each segment. |
| Discovery checkboxes | `role="checkbox"` with `aria-checked` and `aria-label="Positives Beispiel: SS {paragraph} {gesetz}"` / `aria-label="Negatives Beispiel: ..."` |
| Discovery example bar | `role="region"`, `aria-label="Ausgewaehlte Beispiele fuer Entdeckungssuche"`, `aria-live="polite"` for chip additions/removals |
| Discovery chips | Each chip is `<button>` with `aria-label="Beispiel entfernen: SS {paragraph} {gesetz}"` |
| SearchModeToggle (extended) | Existing `role="radiogroup"` pattern extended with 4th segment |
| Graph zoom controls | Visible +/- buttons as alternative to mouse wheel. `aria-label="Vergroessern"` / `aria-label="Verkleinern"` |
| UndoSnackbar | `role="status"`, `aria-live="polite"`, `aria-label="Aktion rueckgaengig machen"`. "Rueckgaengig" button focusable. |

## Responsive Behavior

| Breakpoint | Behavior |
|------------|----------|
| >= 1024px (lg) | Graph + side panel side by side (graph: flex-1, panel: 320px fixed). Discovery example bar horizontal. |
| >= 768px (md) | Graph full width, side panel overlays as slide-in from right. Discovery chips wrap to second line if needed. |
| < 768px (sm) | Graph full width with pinch-zoom. Side panel slides up from bottom (sheet pattern). Discovery chips stack vertically. +/- buttons remain accessible at 44px touch target. Citation tooltips become bottom-sheet on tap. |
| Touch targets | All interactive graph elements: 44px minimum hit area. +/- discovery buttons: 40px height. Citation links: inline but with 8px vertical padding for touch. |
