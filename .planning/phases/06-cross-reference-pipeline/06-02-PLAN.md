    **In backend/src/paragraf/services/qdrant_store.py, add these methods to QdrantStore class:**

    1. `async def set_references_payload(self, point_id: str, references: list[dict]) -> None:`
       - Calls `self.client.set_payload(collection_name=self.collection_name, payload={"references_out": references}, points=[point_id], wait=False)`
       - Deutsche docstring: "Setzt references_out Payload fuer einen Punkt ohne Re-Embedding."

    2. `async def extract_all_references(self, extractor, gesetz_filter: str | None = None, batch_size: int = 100) -> dict[str, int]:`
       - Scrolls through all points (or filtered by gesetz if gesetz_filter provided) using `self.client.scroll()`
       - For each point, calls `extractor.extract(text)` on the text payload
       - Calls `set_references_payload` for each point with non-empty references
       - Uses `wait=False` per point for performance (per Pitfall 3)
       - Returns stats dict: `{"total_points": N, "points_with_refs": M, "total_refs": R}`
       - Scroll loop pattern:
         ```python
         offset = None
         while True:
             records, next_offset = await self.client.scroll(
                 collection_name=self.collection_name,
                 scroll_filter=gesetz_cond,
                 limit=batch_size,
                 offset=offset,
                 with_payload=["text", "gesetz", "paragraph"],
             )
             if not records:
                 break
             for record in records:
                 text = record.payload.get("text", "")
                 refs = extractor.extract(text)
                 if refs:
                     await self.set_references_payload(str(record.id), refs)
             offset = next_offset
             if offset is None:
                 break
         ```

    3. `async def get_outgoing_references(self, gesetz: str, paragraph: str) -> list[dict]:`
       - Scrolls with filter on gesetz + paragraph fields (use existing `_build_filter` pattern with SearchFilter)
       - Returns `record.payload.get("references_out", [])` from matched point
       - If multiple points match (absatz chunks), merge and deduplicate references by (gesetz, paragraph, absatz) tuple

    4. `async def get_incoming_references(self, gesetz: str, paragraph: str, limit: int = 50) -> list[dict]:`
       - Builds NestedCondition filter per Research Pattern 3:
         ```python
         nested_filter = models.Filter(must=[
             models.NestedCondition(nested=models.Nested(
                 key="references_out",
                 filter=models.Filter(must=[
                     models.FieldCondition(key="gesetz", match=models.MatchValue(value=gesetz)),
                     models.FieldCondition(key="paragraph", match=models.MatchValue(value=paragraph)),
                 ]),
             ))
         ])
         ```
       - Scrolls with this filter, returns list of dicts with citing paragraph info: `{"gesetz": ..., "paragraph": ..., "chunk_id": ..., "text_preview": first_200_chars}`

    5. `async def count_incoming_references(self, gesetz: str, paragraph: str) -> int:`
       - Uses same NestedCondition filter as get_incoming_references
       - Calls `self.client.count(collection_name=self.collection_name, count_filter=nested_filter, exact=True)`
       - Returns `count_result.count`

    6. `async def create_reference_indexes(self) -> None:`
       - Creates keyword payload indexes for nested fields to speed up NestedCondition queries (per Pitfall 4):
         ```python
         await self.client.create_payload_index(
             collection_name=self.collection_name,
             field_name="references_out[].gesetz",
             field_schema=models.PayloadSchemaType.KEYWORD,
         )
         await self.client.create_payload_index(
             collection_name=self.collection_name,
             field_name="references_out[].paragraph",
             field_schema=models.PayloadSchemaType.KEYWORD,
         )
         ```
       - Wrap in try/except to handle case where index already exists (idempotent)
