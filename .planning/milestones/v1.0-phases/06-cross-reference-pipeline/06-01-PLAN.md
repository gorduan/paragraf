  <action>
    **Create backend/src/paragraf/services/cross_reference.py:**

    Build a `CrossReferenceExtractor` class with:
    1. Class-level compiled regex patterns built from LAW_REGISTRY keys (sorted by length descending for longest-match-first, per Pitfall 1):
       - `_LAW_PATTERN`: alternation of all escaped LAW_REGISTRY keys, longest first
       - `CITE_SINGLE`: `§\s*(\d+\w?)(?:\s+Abs\.\s*(\d+))?\s+(LAW_PATTERN)` with context prefix
       - `CITE_ARTIKEL`: `Art\.?\s*(\d+\w?)(?:\s+Abs\.\s*(\d+))?\s+(LAW_PATTERN)` with context prefix
       - `CITE_PLURAL`: `§§\s*(\d+\w?)(?:\s*,\s*(\d+\w?))+\s+(LAW_PATTERN)` for comma-separated plural
       - Context prefix pattern: `(?:i\.\s*V\.\s*m\.|gem(?:aess|ae(?:ss|ß)|\.)|nach|siehe|vgl\.)\s+` (optional)

    2. `extract(self, text: str) -> list[dict]` method:
       - Run all patterns against text
       - For each match, build a dict with keys: gesetz, paragraph, absatz, raw, verified, kontext
       - `paragraph` formatted as `"§ {num}"` or `"Art. {num}"`
       - `verified` = `gesetz.strip() in LAW_REGISTRY`
       - `kontext` extracted from context prefix match group (map "gem." -> "gemaess", "vgl." -> "siehe")
       - Deduplicate by (gesetz, paragraph, absatz) tuple

    3. i.V.m. chain handling (per D-03, Pitfall 2):
       - Two-pass approach: first extract all citations normally
       - Then scan for `i.V.m.` pattern between citations
       - If a citation before i.V.m. has no law abbreviation, propagate the law abbreviation from the next citation in the chain
       - Mark the earlier citation's kontext as "i.V.m."

    4. SSSS plural handling (per Pitfall 5):
       - Pattern for `§§ 36, 37 SGB XI` extracts each number as a separate reference
       - All share the same gesetz from the end

    Module-level logger: `logger = logging.getLogger(__name__)`
    Deutsche docstrings on class and all public methods.
    `from __future__ import annotations` at top.

    - Assert on returned list length, field values
    - Additional edge cases: multiple citations in one paragraph, "nach § 7 Abs. 2 SGB IV" context keyword, Art. citations with Abs.
